Git et GitHub - Openclassroom - Marc G Gauthier


Activité 1 : 

Le but de cet exercice est de mettre en pratique les connaissances acquises au cours depuis le début de ce tutoriel.
Les commandes de bases de GIT seront utilisées.
Je peux citer, entre autre :
git clone
git status
git add
git commit (avec ou sans -a)
git log
git push
git pull
etc ...

Ajout de fichier : 

Le fichier "add.html.twig" présente une vue issu d'un projet symfony.
Edit : cette vue est un template Twig. Twig est le moteur de template par défaut du framework PHP Symfony.

Avec le fichier "The Twig Book", documentation officielle de Twig, ce moteur n'aura plus de secrets pour vous !
